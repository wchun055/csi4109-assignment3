# CSI 4109 - Assignment 3
*Implementation of the Yo-Yo algorithm.*

## Setup / Instructions:
1. Clone the repository.

2. Start up your virtual environment. We are using [venv](https://docs.python.org/3/library/venv.html).

    *Note: This is optional, skip to the next step if you do not wish to use a virtual environment.*

   In your terminal, type:

   ```python -m venv venv```
  
4. Install the dependencies. In your terminal, type:

   ```pip install -r requirements.txt```

   Your virtual environment/computer now has the required dependencies.

5. Run the program by typing in the terminal:

   ```python main.py```
